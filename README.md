---

#### [funcoesJS](/funcoesJS) : Nessa pasta, todos os exercícios relacionados às funções estão armazenados 

#### [exerciciosPOO](/exerciciosPOO) : Nessa pasta, todos os exercícios relacionados à POO estão armazenados.

#### [package](/package.json) || [package-lock](/package-lock.json) : Nesses arquivos estão os módulos e configurações usados
